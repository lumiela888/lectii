SEO - Optimizarea motoarelor de căutare Optimizarea motoarelor de căutare (SEO) este procesul de a îmbunătăți clasamentul (vizibilitatea) unui site web în motoarele de căutare. Cu cât un site web este mai mare (sau mai frecvent) este afișat într-o listă de motoare de căutare (cum ar fi Google), cu atât este de așteptat să fie mai mulți vizitatori.
